API Documentation

Integrate our advanced Fake News detection directly into your applications.

Endpoint

POST /predict

Request Body

Send a JSON payload containing the text you want to classify.

{
  "text": "The full headline and article body to analyze..."
}

Response

The API will return a JSON object with the prediction result and the model's confidence level.

{
  "prediction": "REAL" | "FAKE",
  "probability": 0.942
}

Usage Limits

Our current public development endpoint is rate-limited to 100 requests per hour per IP. For scaling and enterprise keys, please contact our support team.
